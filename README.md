# DeepFake Detection

AI-powered DeepFake Detection system developed using Python, Flask, OpenCV, PyTorch, and Librosa to identify manipulated images, videos, and audio files. The project uses deep learning techniques to analyze media content and detect whether the uploaded media is real or fake through an interactive web-based interface.

---

## Features
- DeepFake image detection
- Video frame analysis
- Audio fake voice detection
- AI-powered prediction system
- Flask-based web interface
- Real-time media processing
- Upload and analyze media files easily

---

## Technologies Used
- Python
- Flask
- OpenCV
- PyTorch
- NumPy
- Librosa
- Torchvision
- PIL (Python Imaging Library)
